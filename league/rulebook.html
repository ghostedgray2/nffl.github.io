On-Field Rules

Player Count

The minimum number of players is 12 (6v6). A 6v7, 7v8, and so on, can be forced. If granted permission from a referee or commissioner, a 5v5 or 5v6 can be legal. The player count MUST be agreed on prior to the game in the scheduling thread along with the game time.

Game Re-Schedule

Milking

During the final minute of the second and fourth quarter, you cannot intentionally run the play clock below 10 seconds prior to the play.

Turnover

Gamepasses

Trucking is not allowed under any circumstances. Hitsticking is useless due to the fumble chance being 0%.

Turnover

JP Exploits & Glitches

Angling and other jump power glitches are not allowed. This is because you gain an unfair advantage by using the glitch.

Player Ejection & 10 YD Penalty

Grounding

Grounding is purposefully overthrowing and not throwing near a receiver (usually to avoid pressure). A 10-yard penalty and a loss of a down will result. If the pass hits the stadium's stands, it will be called grounding.

10 YD Penalty

PClocks

Each team gets 1 PClock per drive. You CANNOT PClock mid-drive; you can only PClock before a drive. PClocks may go as long as needed, however if a referee detects you abusing this, there will be a punishment.

5 YD Penalty & Loss of Down

Offensive Linemen

Teams are allowed 1 full time blocker. Full time blockers must be lined up inside the hashes. These players catch and cannot go past the LOS until the ball is thrown. To be declared eligeble, the player cannot block further than 5 yards back.

10 YD Penalty & Loss of Down

4th & 25 Attempts

Teams can elect to go for a 4th & 25 from their own 25 yard line. This must be called prior to the kickoff and the calling team must be losing. Additionally, this can only be called in the 4th quarter. If possible, let the referee set this up with commands.

4th Downs

Teams can elect to go for it (GFI) at ANY time by saying so in chat. To go for it, the following commands should be run: /down 1; /newplay; /down 4.

QB Running

Quarterbacks are not allowed to run the ball while forcing.

10 YD Penalty & Loss of Down

Referee Guidelines

A referee must be present during a game. An emergency ref can be decided upon. Refs must be given maximum admin powers. Purposefully moving or affecting the refs' ability to make accurate calls will result in a 15-yard penalty. If it continues, it will result in an FFL. A ref's private server must be used unless the ref grants permission to use an FO’s server.

Pre-Game Rules

Not playing at an agreed-upon time will result in an FFL. Only 2 walk-ons are allowed, these walk-ons cannot be from another team unless they demand before joining a team. Before the game starts, both FOs must be given admin level 4. Coaches can be given admin level 1 with permission from both FOs. If a referee deems administrative abuse, it will result in a 15-yard penalty.
